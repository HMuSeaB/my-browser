MyBrowser Quick Start

Package contents:
- my-browser.exe
- portable.flag
- README-QuickStart.txt

How to use:
1. Extract the zip to a normal writable folder.
2. Keep portable.flag in the same directory as my-browser.exe.
3. Start my-browser.exe once. Data will be stored in MyBrowserData next to the program.

Before first use:
- Prepare the Camoufox runtime first.
- You can install it with:
  pip install camoufox
  camoufox fetch

Notes:
- This portable package stores profile data locally beside the executable.
- Removing the exe does not remove MyBrowserData automatically.
- If you prefer installed mode, remove portable.flag before first launch.
